.. _simple diffusion application:

Simple Diffusion
================

This is a training application which utilises the majority of the
functionality available in the LFRic Core library. The application is
designed to provide a comprehensive starting point from which to
develop new applications and to learn how LFRic applications work.

The science code itself is a simple horizontal diffusion of a scalar
field.
